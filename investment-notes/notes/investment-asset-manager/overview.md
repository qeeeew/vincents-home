# 투자자산운용사 자격증 개요와 공부 로드맵

투자자산운용사 페이지는 시험 범위 전체를 한 번에 쌓아가는 마크다운 기반 정리노트 홈입니다.  
핵심 개념 요약, 법규 정리, 계산 문제 포인트, 기출 복기 노트를 계속 추가하기 쉽게 만드는 것이 목적입니다.

## 이 페이지에서 쌓아갈 것

- 시험 구조와 과목별 범위
- 자주 나오는 법규와 제도 용어
- 채권, 주식, 파생상품, 펀드 계산 포인트
- 기출 문제에서 반복되는 함정 정리
- 실전 직전 복습용 압축 노트

## 공부 흐름 제안

1. 시험 범위를 큰 단원 기준으로 먼저 훑습니다.
2. 금융상품, 운용전략, 법규를 나눠서 요약 노트를 만듭니다.
3. 계산 문제는 공식을 외우기보다 어떤 상황에서 어떤 식을 꺼내는지 중심으로 정리합니다.
4. 틀린 문제는 별도 마크다운 문서로 빼서 오답 패턴을 축적합니다.
5. 시험 직전에는 압축 노트와 오답노트만 반복합니다.

## 과목 정리 프레임

### 1. 금융시장과 투자상품

- 주식, 채권, 수익증권, 파생상품의 구조를 비교
- 각 상품의 수익 구조와 리스크 요인을 정리
- 계산형 문항에 필요한 기본 공식만 별도 추출

### 2. 자산운용 전략

- 포트폴리오 이론과 성과평가 개념 정리
- 거시경제 지표와 시장 해석 틀 연결
- 액티브 운용과 패시브 운용의 차이 정리

### 3. 관련 법규와 윤리

- 투자자 보호, 불건전 영업행위, 내부통제 관련 포인트 정리
- 숫자 기준, 기간 기준, 금지 행위를 표로 묶기
- 헷갈리는 법 조항은 사례형으로 다시 써보기
